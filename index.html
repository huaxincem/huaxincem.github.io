hx_table

表格使用

api

prop 名称 格式
tableData 表格数据 Array
select 是否显示多选 Boolean
isShowIndex 是否显示索引 Boolean
column 表头列表 Array
tableRowClassName 表格样式 Object
isLazy 是否支持懒 Boolean
childrenText 子节点名称 String,默认值”children”
isHasChildren 懒加载中判断是否有子节点 String,默认值”hasChildren”
rowKey 判断子节点的值 String,Number,默认值”id”
select 是否支持勾选 Boolean,默认值 false

methods

事件 名称 入参
hx-click 单击 row
hx-dblclick 双击 row
hx-select 选择 obj, this.parentIndex
loadTree 选择 tree, treeNode, resolve
